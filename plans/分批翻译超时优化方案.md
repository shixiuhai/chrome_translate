# 分批翻译超时优化方案

## 一、问题分析

### 1.1 当前问题

**问题现象：**
- 全文翻译功能在大页面时仍然超时失败
- 即使设置了60秒超时，内容过大的页面仍然无法完成

**根本原因：**

| 问题 | 位置 | 说明 |
|------|------|------|
| 单次请求过大 | [`content.js:27-48`](content.js:27) | 将整个 `body.innerHTML` 作为单次请求发送 |
| 无分批处理 | [`content.js:8-88`](content.js:8) | 没有将内容拆分成小组分别处理 |
| 无进度反馈 | [`content.js:36`](content.js:36) | 只显示"正在翻译中"，用户不知道进度 |
| 无错误恢复 | [`content.js:82-84`](content.js:82) | 失超时后完全失败，无法部分恢复 |
| 固定超时 | [`background.js:139`](background.js:139) | 所有请求使用30秒固定超时 |

### 1.2 限制条件

- **LibreTranslate API限制**：
  - 单次请求建议最大字符数：约5000字符
  - 建议超时时间：10-15秒/批次
  - 并发请求过多可能导致被限流

- **Chrome扩展限制**：
  - 消息传递有大小限制（但通常足够）
  - 用户需要等待完成，需要进度反馈

## 二、优化方案设计

### 2.1 整体架构

```mermaid
graph TD
    A[用户触发全文翻译) --> B[拆分HTML为可翻译单元]
    B --> C[分组形成翻译批次]
    C --> D[并发翻译批次]
    D --> E[渐进式应用翻译结果]
    E --> F[错误处理与重试]
    
    B --> B1[提取段落/标题/列表项]
    B --> B2[过滤无需翻译内容]
    
    C --> C1[每批次控制字符数]
    C --> C2[每批次控制单元数]
    
    D --> D1[限制并发数量]
    D --> D2[每批次独立超时]
    
    E --> E1[每完成一批立即应用]
    E --> E2[用户可见翻译进度]
    
    F --> F1[失败批次自动重试]
    F --> F2[显示完成统计]
```

### 2.2 HTML拆分策略

#### 2.2.1 翻译单元定义

将页面内容拆分为独立的翻译单元，每个单元包含：

```javascript
{
  id: string,
  type: string,
  originalText: string,
  originalNode: HTMLElement,
  placeholder: string,
  metadata: {
    parentTag: string,
    isLink: boolean,
    hasSpecialChars: boolean
  }
}
```

#### 2.2.2 拆分规则

| 元素类型 | 拆分策略 |
|---------|---------|
| 段落 | 每个p作为一个单元 |
| 标题 | 每个标题作为一个单元 |
| 列表项 | 每个列表项作为一个单元 |
| 表格单元格 | 每个单元格作为一个单元 |
| 按钮 | 按钮文本作为一个单元 |
| 链接文本 | 链接文本作为一个单元 |
| 文本节点 | 独立文本节点作为单元 |

#### 2.2.3 过滤规则

**排除元素：**
- 脚本：`<script>`, `<style>`, `<noscript>`
- 表单元素：`<input>`, `<textarea>`, `<select>`
- 代码块：`<code>`, `<pre>`
- 内嵌内容：`<iframe>`, `<object>`, `<embed>`, `<svg>`, `<canvas>`
- 隐藏元素、可编辑元素

**排除文本：**
- 纯数字、空白文本、纯符号

### 2.4 并发翻译控制

- 使用并发控制器限制同时进行的翻译批次数
- 每个批次独立超时控制
- 失败自动重试机制

### 2.5 进度反馈机制

- 显示翻译进度条
- 实时更新翻译完成百分比
- 显示失败批次数量

### 2.6 翻译结果应用

- 每完成一个批次立即应用到页面
- 用户可以看到翻译进度
- 失败批次不会影响其他批次

## 四、核心函数伪代码

```javascript
// 批次配置
const BATCH_CONFIG = {
  maxCharsPerBatch: 4000,
  maxUnitsPerBatch: 20,
  concurrency: 3,
  batchTimeout: 15000,
  retryTimes: 2,
  retryDelay: 1000
};

// 并发控制器
class ConcurrencyController {
  constructor(maxConcurrency) {
    this.maxConcurrency = maxConcurrency;
    this.running = 0;
    this.queue = [];
  }
  
  async run(task) {
    while (this.running >= this.maxConcurrency) {
      await new Promise(resolve => this.queue.push(resolve));
    }
    this.running++;
    try {
      return await task();
    } finally {
      this.running--;
      if (this.queue.length > 0) this.queue.shift()();
    }
  }
}

// 主翻译函数
async function translateEntirePageWithBatches(source, target) {
  if (isTranslating) return;
  
  isTranslating = true;
  
  try {
    // 1. 拆分HTML为翻译单元
    const units = splitHtmlToUnits();
    if (units.length === 0) {
      showNotification('页面中没有可翻译的内容');
      return;
    }
    
    // 2. 分组形成翻译批次
    const batches = groupUnitsIntoBatches(units, BATCH_CONFIG);
    
    // 3. 显示进度
    showProgress(batches.length);
    
    // 4. 并发翻译
    const controller = new ConcurrencyController(BATCH_CONFIG.concurrency);
    const results = await Promise.all(batches.map((batch, index) =>
      controller.run(() => translateBatchWithRetry(batch, source, target, BATCH_CONFIG))
    ));
    
    // 5. 应用翻译结果
    results.forEach(result => {
      if (result.success) {
        applyTranslationResult(result.data);
      }
    });
    
    // 6. 显示完成状态
    hideProgress();
    showNotification('页面翻译完成！');
  } catch (error) {
    showNotification(`翻译失败: ${error.message}`, 'error');
    hideProgress();
  } finally {
    isTranslating = false;
  }
}

// 拆分HTML为翻译单元
function splitHtmlToUnits() {
  const units = [];
  const walker = document.createTreeWalker(
    document.body,
    NodeFilter.SHOW_ELEMENT | NodeFilter.SHOW_TEXT,
    {
      acceptNode: function(node) {
        if (node.nodeType === Node.ELEMENT_NODE) {
          const tag = node.tagName.toUpperCase();
          const excluded = ['SCRIPT', 'STYLE', 'NOSCRIPT', 'IFRAME', 
                           'OBJECT', 'EMBED', 'INPUT', 'TEXTAREA', 
                           'SELECT', 'CODE', 'PRE', 'CANVAS', 'SVG'];
          if (excluded.includes(tag)) return NodeFilter.FILTER_REJECT;
          if (node.hidden || node.style.display === 'none') return NodeFilter.FILTER_REJECT;
          if (node.isContentEditable) return NodeFilter.FILTER_REJECT;
        }
        return NodeFilter.FILTER_ACCEPT;
      }
    }
  );
  
  let node;
  while (node = walker.nextNode()) {
    if (node.nodeType === Node.TEXT_NODE) {
      const text = node.textContent.trim();
      if (text.length >= 2 && !isExcludedText(text)) {
        units.push({
          id: `unit-${Math.random().toString(36).substr(2, 9)}`,
          type: node.parentNode?.tagName?.toLowerCase() || 'text',
          originalText: text,
          originalNode: node
        });
      }
    } else if (node.nodeType === Node.ELEMENT_NODE) {
      const tag = node.tagName.toUpperCase();
      const inlineTags = ['BUTTON', 'A', 'SPAN', 'B', 'I', 'STRONG', 'EM'];
      if (inlineTags.includes(tag) && node.textContent.trim().length >= 2) {
        if (!units.some(u => u.originalNode === node || u.originalNode.parentNode === node)) {
          units.push({
            id: `unit-${Math.random().toString(36).substr(2, 9)}`,
            type: tag.toLowerCase(),
            originalText: node.textContent.trim(),
            originalNode: node
          });
        }
      }
    }
  }
  
  return units;
}

// 分组形成翻译批次
function groupUnitsIntoBatches(units, config) {
  const batches = [];
  let currentBatch = [];
  let currentChars = 0;
  
  for (const unit of units) {
    const chars = unit.originalText.length;
    
    if (chars > config.maxCharsPerBatch) {
      if (currentBatch.length > 0) {
        batches.push({ units: currentBatch, totalChars: currentChars });
        currentBatch = [];
        currentChars = 0;
      }
      batches.push({ units: [unit], totalChars: chars });
      continue;
    }
    
    if (currentChars + chars > config.maxCharsPerBatch || 
        currentBatch.length >= config.maxUnitsPerBatch) {
      if (currentBatch.length > 0) {
        batches.push({ units: currentBatch, totalChars: currentChars });
      }
      currentBatch = [unit];
      currentChars = chars;
    } else {
      currentBatch.push(unit);
      currentChars += chars;
    }
  }
  
  if (currentBatch.length > 0) {
    batches.push({ units: currentBatch, totalChars: currentChars });
  }
  
  return batches;
}

// 批次翻译带重试
async function translateBatchWithRetry(batch, source, target, config) {
  let retries = 0;
  let lastError = null;
  
  while (retries <= config.retryTimes) {
    try {
      const texts = batch.units.map(u => u.originalText);
      const response = await chrome.runtime.sendMessage({
        action: 'translate',
        data: { q: texts, source, target, format: 'text' }
      });
      
      if (!response.success) {
        throw new Error(response.error || 'Translation failed');
      }
      
      const translations = response.data.translatedText || [];
      return {
        success: true,
        data: batch.units.map((unit, i) => ({
          unit,
          translatedText: translations[i] || unit.originalText
        }))
      };
    } catch (error) {
      lastError = error;
      retries++;
      if (retries <= config.retryTimes) {
        await new Promise(r => setTimeout(r, config.retryDelay * retries));
      }
    }
  }
  
  return { success: false, error: lastError?.message || 'Unknown error' };
}

// 应用翻译结果
function applyTranslationResult(result) {
  result.forEach(item => {
    try {
      const { unit, translatedText } = item;
      if (!document.body.contains(unit.originalNode)) return;
      
      if (unit.type === 'text') {
        unit.originalNode.textContent = translatedText;
      } else {
        const originalText = unit.originalText;
        unit.originalNode.innerHTML = unit.originalNode.innerHTML.replace(
          originalText,
          translatedText
        );
      }
    } catch (error) {
      console.error('Failed to apply translation:', error);
    }
  });
 });
}

function isExcludedText(text) {
  if (/^\d+([,.]\d+)*$/.test(text)) return true;
  if (/^[^\w\s\u4e00-\u9fff]+$/.test(text)) return true;
  return false;
}
```
